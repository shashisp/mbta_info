# mbta_info
Massachusetts Bay Transportation Authority Rail departure board

### Instructions

###### Create virutal environment
```
virtualenv env
source env/bin/activate
```
###### Install Dependencies
```
pip install -r requirements.txt
```
### run the project
```
python manage.py runserver
```
